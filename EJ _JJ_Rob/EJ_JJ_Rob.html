You can make your own DJ box with Arduino!

By using Arduino, we can becomes DJs. We can change light and sounds by using a touch sensor. .

Arduino is an open-source electronic platform that can detect sound and temperature. Arduino is easy to use and has high accessibility so it can be used in many ways. In this project, we used motion sense to change songs and lights.
